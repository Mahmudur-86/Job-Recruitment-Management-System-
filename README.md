It's a Project where the actor is Jobseeker and Admin.
Here jobseeker can apply for a specific post, by default his/her application pending.Admin can see the jobseeker CV and decide  whether he/she applicable for this post.
If he/he approved and admin will send the interview question (MCQ type) and also send a mail to jobseeker .Jobseeker answer the all question at an one time.
After receiving the interview answer and admin will send a mail jobseeker whether he is pass or write the next process for post.
If Jobseeker pass the interview exam and Admin send a mail. Admin will tell the jobseeker to eait for the recruitment letter.
Admin prepare recruitment letter for the jobseeker and also send jobseeker email as a pdf.
In this project, there will be automation type report generate at admin panel.
